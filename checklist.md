# 🔐 Privacy Hardening Checklist

## 🌐 1. Browser Security
- [ ] Disable Third-Party Cookies
- [ ] Use uBlock Origin to block ads & trackers
- [ ] Install Privacy Badger
- [ ] Use Private Browsing when needed

---

## 📡 2. Header Protection
- [ ] Set Referrer-Policy: no-referrer
- [ ] Use <meta name="referrer" content="origin">
- [ ] Avoid leaking full URLs in requests

---

## 🌍 3. Network Privacy
- [ ] Enable DNS over HTTPS (DoH)
- [ ] Use a trusted VPN
- [ ] Use Tor Browser for maximum anonymity

---

## 🍪 4. Cookie Management
- [ ] Regularly clear cookies
- [ ] Block third-party cookies
- [ ] Monitor long-lived tracking cookies (max-age)

---

## 📍 5. Device Permissions
- [ ] Disable location for unnecessary apps
- [ ] Turn off precise location
- [ ] Review camera & microphone access

---

## ⚠️ Reminder
No system provides 100% privacy, but layered protection reduces tracking.
